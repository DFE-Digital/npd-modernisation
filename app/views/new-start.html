Find and explore data in the National Pupil Database

Welcome to the Find and explore data in the National Pupil Database tool.

Anyone can use it to find what sorts of data the Department holds in the National Pupil Database (NPD).

The NPD is the Department’s store of data it has collected on education. It is collected at various points in the year, each distinct dataset covering an aspect of education and childcare, sometimes coming in from external organisations (exam bodies for example).

It allows the Department to inform policy and assess how it is performing. If you are a researcher who would like to apply to use data within the NPD, please find the guidance here.

How to use the tool

There are two ways you can explore the type of data the NPD holds:

Browsing via categories

We have arranged the data items (think of these as the actual bits of data we collect from educational establishments) into concepts. Each concept holds data items that have the same core value: so the concept ‘Authorised absence due to study leave’ contains all the data items we collect on that subject – 6 in total.

Each concept is itself placed in a hierarchy of categories – these are containers that help you browse, logically, to the concept you’re interested in. The top four categories are: Attainment, Demographics, Absence and Exclusion, and Children in need and children looked after. You can also find concepts using a simple text search. These methods should be particularly useful if you are new to the NPD.

Browsing via datasets

You can also choose to view the data-items as we hold them in the NPD. This is as 23 lists of data items based on DFE collections – for example the School Census. This method might be more useful if you are an experienced user of the NPD.

Saving a list of favourite data items

If you’d like to keep a list of the data items you are interested in you can use the ‘My List’ function by ticking data items you’d like to copy to your device’s temporary memory and selecting ‘Save to my list’.
