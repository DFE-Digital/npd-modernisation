Apply for National Pupil Database data This service provides access to individual level data owned by DfE to undertake a research project within the ONS Secure Research Service.

We will ask for information on the following;

  • You and your organisation
  • The researchers you'll be working with
  • The type of research project you need the data for
  • The data you require
  • How you will process the data
  • How long you'll need the data for

Before you start

You'll need the following before you'll be allowed access to the data

  • A current basic DBS check
  • An ONS accredited researcher account
  • To have applied for access to the ONS Secure Research Service
